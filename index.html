Summary: Latitude and Observed Weather



The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.

fter assembling the dataset, we used Matplotlib to plot various aspects of weather vs. latitude. Factors examined included: temperatue, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis as well as explanations and descriptions of any trends and correlations witnessed.
